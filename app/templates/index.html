Heroku

What is Heroku?
Heroku is a platform as a service (PaaS) that allows developers to deploy, manage, and scale modern applications.

What does this application do?
This app is an example of deployment on Heroku, using Flask for the backend, HTML/CSS for the frontend, and an SQLite database.

functionality:

  • Loads a page with basic information.
  • Simple connection to the database.
  • Containerized and deployed automatically.
